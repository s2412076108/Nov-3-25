# Student Marks Program - Clean Code & Git Assignment

## Overview
This project refactors a messy C program into a clean, readable, and modular version following Clean Code principles.  
It also demonstrates how to manage code using Git version control.

---

## Steps Followed

1. **Added messy version of code**
2. **Fixed indentation and structure**
3. **Renamed variables for clarity**
4. **Added modular functions**
5. **Added comments and formatted output**

---

## Git Commands Used
```bash
git init
git add .
git commit -m "commit message"
git log --oneline
git remote add origin https://github.com/<username>/clean-code-assignment.git
git branch -M main
git push -u origin main
